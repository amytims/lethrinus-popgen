# lethrinus-popgen

A collection of scripts to run data processing and variant calling on Lethrinus punctulatus data

1. Raw short-read data QC (`fastqc`) and aggregation of QC reports (`multiqc`)
2. Filtering data to remove adapters and low-quality bases (`Trimmomatic`)
3. QC on filtered short read data (`fastqc`) and aggregation of QC reports (`multiqc`)
